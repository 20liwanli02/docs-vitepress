# 第8章：A2A协议

## 一、A2A 概述

### 1.1 什么是 A2A？

**A2A（Agent-to-Agent）** 是一种 Agent 间通信协议，允许不同的 AI Agent 相互协作完成任务。

```
┌─────────────────────────────────────────────────────────────┐
│                      A2A 架构                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│     ┌─────────────┐                    ┌─────────────┐     │
│     │  主 Agent   │                    │ 远程 Agent  │     │
│     │  (调用者)    │ ◄────────────────► │  (被调用者)  │     │
│     └─────────────┘                    └─────────────┘     │
│            │                                  │            │
│            │         A2A 协议                 │            │
│            │    ┌──────────────────┐         │            │
│            └───►│  Agent Card      │◄────────┘            │
│                 │  Message/Send    │                      │
│                 │  Tasks           │                      │
│                 └──────────────────┘                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 A2A 核心概念

| 概念 | 说明 |
|-----|------|
| **Agent Card** | Agent 的"名片"，描述能力、端点等 |
| **Message/Send** | 发送消息给远程 Agent |
| **Tasks** | 任务管理（创建、查询、取消） |
| **Skills** | Agent 的技能列表 |

### 1.3 A2A vs MCP

| 对比项 | A2A | MCP |
|-------|-----|-----|
| 通信对象 | Agent 之间 | Agent 与工具 |
| 复杂度 | 更复杂 | 相对简单 |
| 自主性 | 远程 Agent 自主决策 | 工具被动执行 |
| 适用场景 | 多 Agent 协作 | 工具集成 |

### 2.2 A2AClientManager 实现

```python
# api/app/domain/services/tools/a2a.py

class A2AClientManager:
    """A2A 客户端管理器"""
    
    def __init__(self, a2a_config: A2AConfig):
        self._a2a_config = a2a_config
        self._exit_stack = AsyncExitStack()
        self._httpx_client: Optional[httpx.AsyncClient] = None
        self._agent_cards: Dict[str, Any] = {}  # Agent 卡片缓存
        self._initialized = False
    
    async def initialize(self):
        """初始化 A2A 客户端"""
        if self._initialized:
            return
        
        # 创建 HTTP 客户端
        self._httpx_client = await self._exit_stack.enter_async_context(
            httpx.AsyncClient(timeout=600)
        )
        
        # 获取所有 Agent 卡片
        await self._get_a2a_agent_cards()
        self._initialized = True
    
    async def _get_a2a_agent_cards(self):
        """获取所有 A2A Agent 的卡片信息"""
        for server_config in self._a2a_config.a2a_servers:
            try:
                # 获取 Agent Card
                response = await self._httpx_client.get(
                    f"{server_config.base_url}/.well-known/agent-card.json"
                )
                response.raise_for_status()
                agent_card = response.json()
                
                # 缓存卡片
                agent_card["enabled"] = server_config.enabled
                self._agent_cards[server_config.id] = agent_card
                
            except Exception as e:
                logger.warning(f"获取 Agent [{server_config.id}] 卡片失败: {e}")
```

### 2.4 调用远程 Agent

```python
class A2AClientManager:
    
    async def invoke(self, agent_id: str, query: str) -> ToolResult:
        """调用远程 Agent"""
        if agent_id not in self._agent_cards:
            return ToolResult(success=False, message="远程 Agent 不存在")
        
        agent_card = self._agent_cards.get(agent_id)
        url = agent_card.get("url", "")
        
        if not url:
            return ToolResult(success=False, message="远程 Agent 端点不存在")
        
        try:
            # 发送 JSON-RPC 请求
            response = await self._httpx_client.post(
                url,
                json={
                    "id": str(uuid.uuid4()),
                    "jsonrpc": "2.0",
                    "method": "message/send",
                    "params": {
                        "message": {
                            "messageId": str(uuid.uuid4()),
                            "role": "user",
                            "parts": [
                                {"kind": "text", "text": query}
                            ]
                        }
                    }
                }
            )
            response.raise_for_status()
            result = response.json()
            
            return ToolResult(
                success=True,
                message="调用远程 Agent 成功",
                data=result
            )
            
        except Exception as e:
            return ToolResult(
                success=False,
                message=f"调用远程 Agent 失败: {str(e)}"
            )
```

## 三、A2ATool 封装

### 3.1 A2ATool 类

```python
class A2ATool(BaseTool):
    """A2A 工具包"""
    
    name: str = "a2a"
    
    def __init__(self):
        super().__init__()
        self._initialized = False
        self.manager: A2AClientManager = None
    
    async def initialize(self, a2a_config: A2AConfig):
        """初始化 A2A 工具"""
        if not self._initialized:
            self.manager = A2AClientManager(a2a_config)
            await self.manager.initialize()
            self._initialized = True
    
    @tool(
        name="get_remote_agent_cards",
        description="获取可远程调用的 Agent 卡片信息列表",
        parameters={},
        required=[]
    )
    async def get_remote_agent_cards(self) -> ToolResult:
        """获取远程 Agent 卡片列表"""
        agent_cards = []
        for id, card in self.manager.agent_cards.items():
            agent_cards.append({
                "id": id,
                **card
            })
        
        return ToolResult(
            success=True,
            message="获取 Agent 卡片列表成功",
            data=agent_cards
        )
    
    @tool(
        name="call_remote_agent",
        description="调用远程 Agent 完成任务",
        parameters={
            "id": {
                "type": "string",
                "description": "远程 Agent 的 ID"
            },
            "query": {
                "type": "string",
                "description": "需要远程 Agent 完成的任务描述"
            }
        },
        required=["id", "query"]
    )
    async def call_remote_agent(self, id: str, query: str) -> ToolResult:
        """调用远程 Agent"""
        return await self.manager.invoke(agent_id=id, query=query)
```

## 六、A2A 最佳实践

### 6.1 设计原则

| 原则 | 说明 |
|-----|------|
| **单一职责** | 每个 Agent 专注一个领域 |
| **清晰描述** | Agent Card 描述要详细 |
| **错误处理** | 处理远程调用失败情况 |
| **超时控制** | 设置合理的超时时间 |

## 七、总结

### 7.1 A2A 优势

| 优势 | 说明 |
|-----|------|
| **专业分工** | 每个 Agent 专注擅长领域 |
| **可扩展** | 轻松添加新的远程 Agent |
| **解耦** | Agent 之间独立部署 |
| **复用** | 同一个 Agent 可被多个主 Agent 调用 |

|-----|---------|
