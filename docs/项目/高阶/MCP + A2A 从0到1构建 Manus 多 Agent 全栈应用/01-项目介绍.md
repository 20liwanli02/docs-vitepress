# 第1章：项目介绍

## 一、项目概述

### 1.1 什么是 MoocManus？

**MoocManus** 是一个**通用 AI Agent 系统**，支持完全私有化部署。它能够：

- 🤖 **智能对话**：支持多轮对话、流式响应、上下文记忆
- 📋 **任务规划**：自动将复杂任务拆解为子步骤，动态调整执行计划
- 🛠️ **工具调用**：支持浏览器自动化、Shell命令、文件操作、网络搜索等
- 🔌 **协议支持**：集成 MCP（模型上下文协议）和 A2A（Agent-to-Agent）协议
- 🐳 **沙箱隔离**：在 Docker 容器中安全执行任务
- 👀 **实时预览**：通过 VNC 实时查看浏览器操作过程

### 1.2 项目演示

#### 场景一：自动搜索整理文档

```
用户：帮我搜索 Python 教程并整理成 Markdown 文档

Agent 执行过程：
1. [规划] 拆解任务为 4 个子步骤
2. [执行] 调用搜索工具搜索 "Python 教程"
3. [执行] 访问搜索结果，提取优质内容
4. [执行] 整理内容，生成 Markdown 文件
5. [完成] 返回文档给用户
```

#### 场景二：网页自动化操作

```
用户：帮我登录 GitHub 并查看今天的 Trending

Agent 执行过程：
1. [规划] 拆解任务为 3 个子步骤
2. [执行] 打开浏览器，导航到 GitHub
3. [执行] 点击 Trending 页面
4. [执行] 截图并提取项目列表
5. [完成] 返回 Trending 列表给用户
```


## 五、应用场景

### 5.1 典型场景

| 场景          | 描述             | 使用工具                    |
| ----------- | -------------- | ----------------------- |
| 📝 **文档处理** | 搜索资料、整理文档、生成报告 | Search + File + Browser |
