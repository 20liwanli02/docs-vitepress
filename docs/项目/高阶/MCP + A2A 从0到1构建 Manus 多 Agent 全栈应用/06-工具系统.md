# 第6章：工具系统

## 一、工具系统概述

### 1.1 什么是工具？

在 Agent 系统中，**工具（Tool）** 是 Agent 与外部世界交互的桥梁。

```
┌─────────────────────────────────────────────────────────────┐
│                      工具的作用                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│     ┌─────────────┐                                         │
│     │   Agent     │                                         │
│     │   (大脑)    │                                         │
│     └──────┬──────┘                                         │
│            │                                                │
│            │ "我需要搜索信息"                                │
│            ▼                                                │
│     ┌─────────────┐                                         │
│     │ SearchTool  │ ◄─── 工具：执行搜索                      │
│     └──────┬──────┘                                         │
│            │                                                │
│            │ 返回搜索结果                                    │
│            ▼                                                │
│     ┌─────────────┐                                         │
│     │   Agent     │                                         │
│     │  (处理结果) │                                         │
│     └─────────────┘                                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 工具类型

| 工具类型 | 功能 | 示例 |
|---------|------|------|
| **BrowserTool** | 浏览器自动化 | 导航、点击、输入、截图 |
| **ShellTool** | 命令执行 | 运行脚本、安装软件 |
| **FileTool** | 文件操作 | 读写文件、管理目录 |
| **SearchTool** | 网络搜索 | Bing 搜索 |
| **MCPTool** | MCP 工具 | 外部工具生态 |
| **A2ATool** | 远程 Agent | 调用其他 Agent |
| **MessageTool** | 人机交互 | 向用户提问 |

## 二、工具抽象设计

### 2.1 BaseTool 基类

```python
class BaseTool(ABC):
    """工具基类"""
    
    name: str  # 工具名称
    
    def get_tools(self) -> List[Dict[str, Any]]:
        """获取工具列表 Schema"""
        tools = []
        for method_name in dir(self):
            method = getattr(self, method_name)
            if hasattr(method, '_tool_schema'):
                tools.append(method._tool_schema)
        return tools
    
    def has_tool(self, tool_name: str) -> bool:
        """判断是否包含指定工具"""
        for tool in self.get_tools():
            if tool['function']['name'] == tool_name:
                return True
        return False
    
    async def invoke(self, tool_name: str, **kwargs) -> ToolResult:
        """调用工具"""
        method = getattr(self, tool_name, None)
        if method and hasattr(method, '_tool_schema'):
            return await method(**kwargs)
        raise ValueError(f"工具 {tool_name} 不存在")
```

### 2.2 工具装饰器

```python
def tool(
    name: str,
    description: str,
    parameters: Dict[str, Any],
    required: List[str]
):
    """
    工具注册装饰器
    
    Args:
        name: 工具名称
        description: 工具描述
        parameters: 参数定义
        required: 必填参数列表
    """
    def decorator(func):
        func._tool_schema = {
            "type": "function",
            "function": {
                "name": name,
                "description": description,
                "parameters": {
                    "type": "object",
                    "properties": parameters,
                    "required": required,
                }
            }
        }
        return func
    return decorator
```

### 2.3 ToolResult 结构

```python
class ToolResult(BaseModel):
    """工具执行结果"""
    
    success: bool = True           # 是否成功
    message: str = ""              # 消息
    data: Optional[Any] = None     # 数据
    
    # 使用示例
    # 成功：ToolResult(success=True, data="搜索结果...")
    # 失败：ToolResult(success=False, message="搜索失败")
```

## 三、内置工具详解

### 3.1 BrowserTool（浏览器工具）

**功能**：浏览器自动化操作

```python
class BrowserTool(BaseTool):
    name: str = "browser"
    
    def __init__(self, browser: Browser):
        self.browser = browser
    
    @tool(
        name="browser_view",
        description="查看当前浏览器页面内容",
        parameters={},
        required=[]
    )
    async def browser_view(self) -> ToolResult:
        return await self.browser.view_page()
    
    @tool(
        name="browser_navigate",
        description="导航到指定URL",
        parameters={
            "url": {
                "type": "string",
                "description": "要访问的URL"
            }
        },
        required=["url"]
    )
    async def browser_navigate(self, url: str) -> ToolResult:
        return await self.browser.navigate(url)
    
    @tool(
        name="browser_click",
        description="点击页面元素",
        parameters={
            "index": {"type": "integer", "description": "元素索引"},
            "coordinate_x": {"type": "number", "description": "X坐标"},
            "coordinate_y": {"type": "number", "description": "Y坐标"}
        },
        required=[]
    )
    async def browser_click(
        self,
        index: Optional[int] = None,
        coordinate_x: Optional[float] = None,
        coordinate_y: Optional[float] = None,
    ) -> ToolResult:
        return await self.browser.click(index, coordinate_x, coordinate_y)
    
    @tool(
        name="browser_input",
        description="在输入框中输入文本",
        parameters={
            "text": {"type": "string", "description": "输入文本"},
            "press_enter": {"type": "boolean", "description": "是否按回车"},
            "index": {"type": "integer", "description": "元素索引"}
        },
        required=["text", "press_enter"]
    )
    async def browser_input(
        self,
        text: str,
        press_enter: bool,
        index: Optional[int] = None,
    ) -> ToolResult:
        return await self.browser.input(text, press_enter, index)
    
    # ... 更多工具方法
```

### 3.2 ShellTool（命令工具）

**功能**：在沙箱中执行 Shell 命令

```python
class ShellTool(BaseTool):
    name: str = "shell"
    
    def __init__(self, sandbox: Sandbox):
        self.sandbox = sandbox
    
    @tool(
        name="shell_exec",
        description="执行Shell命令",
        parameters={
            "command": {
                "type": "string",
                "description": "要执行的命令"
            },
            "timeout": {
                "type": "integer",
                "description": "超时时间（秒）"
            }
        },
        required=["command"]
    )
    async def shell_exec(
        self,
        command: str,
        timeout: int = 30
    ) -> ToolResult:
        result = await self.sandbox.exec_command(command, timeout)
        return ToolResult(
            success=result.exit_code == 0,
            data=result.output,
            message=result.error if result.error else ""
        )
```

### 3.3 FileTool（文件工具）

**功能**：文件读写操作

```python
class FileTool(BaseTool):
    name: str = "file"
    
    @tool(
        name="file_read",
        description="读取文件内容",
        parameters={
            "path": {
                "type": "string",

            }
        },
        required=["path"]

        name="file_write",
        description="写入文件",
        parameters={
            "path": {"type": "string", "description": "文件路径"},
            "content": {"type": "string", "description": "文件内容"}
        },
        required=["path", "content"]
    )
    async def file_write(self, path: str, content: str) -> ToolResult:
        await self.sandbox.write_file(path, content)
        return ToolResult(success=True, message="文件写入成功")
    
    @tool(
        name="file_list",
        description="列出目录内容",
        parameters={
            "path": {"type": "string", "description": "目录路径"}
        },
        required=["path"]
    )
    async def file_list(self, path: str) -> ToolResult:
        files = await self.sandbox.list_dir(path)
        return ToolResult(success=True, data=files)
```

## 四、工具调用流程

### 4.1 完整调用流程

```
┌─────────────────────────────────────────────────────────────┐
│                    工具调用流程                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. LLM 决定调用工具                                         │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ LLM 响应:                                            │   │
│  │ {                                                    │   │
│  │   "tool_calls": [{                                  │   │
│  │     "id": "call_xxx",                               │   │
│  │     "function": {                                   │   │
│  │       "name": "browser_navigate",                   │   │
│  │       "arguments": '{"url": "https://..."}'         │   │
│  │     }                                               │   │
│  │   }]                                                │   │
│  │ }                                                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                           │                                 │
│                           ▼                                 │
│  2. Agent 解析工具调用                                       │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ tool_call_id = "call_xxx"                           │   │
│  │ function_name = "browser_navigate"                  │   │
│  │ function_args = {"url": "https://..."}              │   │
│  └─────────────────────────────────────────────────────┘   │
│                           │                                 │
│                           ▼                                 │
│  3. 查找对应工具                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ tool = self._get_tool("browser_navigate")           │   │
│  │ # 返回 BrowserTool 实例                             │   │
│  └─────────────────────────────────────────────────────┘   │
│                           │                                 │
│                           ▼                                 │
│  4. 发送调用中事件                                           │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ yield ToolEvent(                                    │   │
│  │     tool_call_id="call_xxx",                        │   │
│  │     tool_name="browser",                            │   │
│  │     function_name="browser_navigate",               │   │
│  │     function_args={"url": "https://..."},           │   │
│  │     status=ToolEventStatus.CALLING                  │   │
│  │ )                                                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                           │                                 │
│                           ▼                                 │
│  5. 执行工具                                                 │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ result = await self._invoke_tool(                   │   │
│  │     tool,                                           │   │
│  │     "browser_navigate",                             │   │
│  │     {"url": "https://..."}                          │   │
│  │ )                                                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                           │                                 │
│                           ▼                                 │
│  6. 发送调用完成事件                                         │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ yield ToolEvent(                                    │   │
│  │     tool_call_id="call_xxx",                        │   │
│  │     function_name="browser_navigate",               │   │
│  │     function_result=result,                         │   │
│  │     status=ToolEventStatus.CALLED                   │   │
│  │ )                                                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                           │                                 │
│                           ▼                                 │
│  7. 将结果添加到消息列表                                     │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ tool_messages.append({                              │   │
│  │     "role": "tool",                                 │   │
│  │     "tool_call_id": "call_xxx",                     │   │
│  │     "content": result.model_dump_json()             │   │
│  │ })                                                  │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 五、自定义工具开发

### 5.1 开发步骤

```
1. 创建工具类，继承 BaseTool
2. 使用 @tool 装饰器定义工具方法
3. 在 Flow 中注册工具
```

### 5.2 示例：天气查询工具

```python
# api/app/domain/services/tools/weather.py

from typing import Dict, Any
from app.domain.models.tool_result import ToolResult
from .base import BaseTool, tool

class WeatherTool(BaseTool):
    """天气查询工具"""
    
    name: str = "weather"
    
    def __init__(self, api_key: str):
        self.api_key = api_key
    
    @tool(
        name="weather_get",
        description="获取指定城市的天气信息",
        parameters={
            "city": {
                "type": "string",
                "description": "城市名称，如：北京、上海"
            },
            "unit": {
                "type": "string",
                "description": "温度单位，celsius（摄氏）或 fahrenheit（华氏）",
                "enum": ["celsius", "fahrenheit"]
            }
        },
        required=["city"]
    )
    async def weather_get(
        self,
        city: str,
        unit: str = "celsius"
    ) -> ToolResult:
        """获取天气信息"""
        try:
            # 调用天气 API
            weather_data = await self._fetch_weather(city, unit)
            
            return ToolResult(
                success=True,
                data={
                    "city": city,
                    "temperature": weather_data["temp"],
                    "condition": weather_data["condition"],
                    "humidity": weather_data["humidity"],
                }
            )
        except Exception as e:
            return ToolResult(
                success=False,
                message=f"获取天气失败: {str(e)}"
            )
    
    async def _fetch_weather(self, city: str, unit: str) -> Dict:
        """调用天气 API"""
        import httpx
        
        async with httpx.AsyncClient() as client:
            response = await client.get(
                "https://api.weather.com/v1/current",
                params={
                    "city": city,
                    "unit": unit,
                    "key": self.api_key
                }
            )
            return response.json()
```

## 六、工具最佳实践

### 6.1 工具设计原则

| 原则 | 说明 |
|-----|------|
| **单一职责** | 每个工具只做一件事 |
| **清晰描述** | 描述要详细，让 LLM 理解何时使用 |
| **参数验证** | 验证参数类型和范围 |
| **错误处理** | 返回有意义的错误信息 |
| **幂等性** | 相同输入产生相同结果 |

### 6.2 描述编写技巧

```python
# 好的描述示例
@tool(
    name="browser_navigate",
    description="""
    将浏览器导航至指定网址。
    
    使用场景：
    - 需要访问新网页时
    - 需要打开特定网站时
    
    注意：
    - URL 必须包含协议前缀（如 https://）
    - 如果页面加载失败会返回错误信息
    """,
    ...
)

# 不好的描述示例
@tool(
    name="browser_navigate",
    description="导航",  # 太简单，LLM 无法理解
    ...
)
```

### 6.3 错误处理示例

```python
@tool(...)
async def file_read(self, path: str) -> ToolResult:
    try:
        # 参数验证
        if not path:
            return ToolResult(
                success=False,
                message="文件路径不能为空"
            )
        
        # 执行操作
        content = await self.sandbox.read_file(path)
        
        return ToolResult(success=True, data=content)
        
    except FileNotFoundError:
        return ToolResult(
            success=False,
            message=f"文件不存在: {path}"
        )
    except PermissionError:
        return ToolResult(
            success=False,
            message=f"无权限访问文件: {path}"
        )
    except Exception as e:
        return ToolResult(
            success=False,
            message=f"读取文件失败: {str(e)}"
        )
```
