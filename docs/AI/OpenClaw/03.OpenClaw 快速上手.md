# OpenClaw 快速上手

## 一、第一次对话

### 启动方式

| 部署方式     | 访问方法                                                     |
| ------------ | ------------------------------------------------------------ |
| **云端部署** | `http://服务器IP:18789/?token=你的token` 或 直接通过 QQ/企微/飞书/钉钉发消息 |
| **本地部署** | 先检查 Gateway 状态，再访问 `http://127.0.0.1:18789/?token=你的token` |

**检查 Gateway 状态**：
```bash
openclaw channels status  # 应显示 Gateway reachable
```

### 验证成功标志
- ✅ 收到 AI 回复
- ✅ Web UI 右上角显示模型名称
- ✅ 显示 Token 使用情况

### 提问技巧

| ❌ 不好的提问 | ✅ 好的提问                                               |
| ------------ | -------------------------------------------------------- |
| "帮我"       | "帮我找一下电脑上所有的 PDF 文件，按修改时间排序"        |

1. **明确目标** - 说清楚想做什么
2. **提供上下文** - 给出必要背景信息
3. **具体细节** - 说明具体要求
4. **分步骤** - 复杂任务分步骤说明

## 三、人设配置技巧（Persona）

### 什么是人设
给 AI 设定的**性格、身份和行为方式**，让 AI 更符合你的喜好和工作风格。

### 核心配置文件（Workspace）

位于 `~/.openclaw/workspace/`：

| 文件         | 作用                          | 新手建议   |
| ------------ | ----------------------------- | ---------- |
| **SOUL.md**  | AI 的性格、说话风格、行为准则 | ⭐ 必须配置 |
| **USER.md**  | 用户信息、习惯、偏好          | ⭐ 必须配置 |
| AGENTS.md    | Agent 的工作指令和任务说明    | 进阶使用   |
| MEMORY.md    | AI 的长期记忆和学习内容       | 进阶使用   |
| HEARTBEAT.md | 定期检查和维护任务            | 进阶使用   |
| IDENTITY.md  | AI 的名称、身份、主题设定     | 可选       |
| BOOT.md      | 启动时的初始化配置            | 可选       |

### 配置方法

#### 方法一：Web UI（推荐）
1. 打开 OpenClaw 页面 → 点击 **Agent → Files**
2. 编辑 `SOUL.md` 和 `USER.md`
3. 保存后立即生效，无需重启

#### 方法二：命令行（进阶）
```bash
# 编辑 SOUL.md
nano ~/.openclaw/workspace/SOUL.md

# 编辑 USER.md
nano ~/.openclaw/workspace/USER.md
```

### SOUL.md 配置示例

```markdown
_你不是聊天机器人。你是个靠谱的助手。_

## 核心原则

**认真做事，别敷衍。** 少说"好的！马上帮您处理！"，直接动手。行动比客套话管用。

**有自己的想法。** 可以不同意，可以有偏好，可以觉得有些东西不太行。没个性的助手就是个低效搜索引擎。

**先自己想办法。** 翻文件、看上下文、搜一搜。实在搞不定再问。目标是带答案回来，不是带问题回来。

## 边界

- **密码永远不说出来。** 看到密码、API key、token，闭嘴就好。
- 隐私的事保密。私人聊天、财务信息，看到了当没看到。
- 不确定的对外操作，先问。
- 别发半成品消息到聊天软件。
- 群聊里别乱说话，你不是用户的代言人。

## 风格

该简洁时简洁，该详细时详细。像个靠谱同事说话，不是客服机器人。可以直接，可以有态度，可以指出问题。不拍马屁。不当应声虫。就做个靠谱的。

## 记忆

每次对话你都是新的。这些文件就是你的记忆。读它们，更新它们。
```

- **Name:** Maynor

### 风格调优技巧

**如果 AI 太啰嗦**：
```markdown

- 简洁明了，不超过3句话
- 直接给出答案，不解释过程
- 只在必要时提供详细信息
```

**如果 AI 太冷淡**：
```markdown
## 风格
- 友好热情
- 适当使用emoji
- 关心用户感受
```

**如果 AI 不够专业**：
```markdown
## 风格
- 专业准确
- 提供数据支持
- 引用可靠来源
```

### 快速配置模型（命令行向导）

```bash
openclaw onboard
```

**配置步骤**：
1. **选择初始化模式** → 快速开始
2. **处理现有配置** → 使用现有值/重新配置
3. **选择模型提供商**（方向键选择，空格确认）：
   - Anthropic（Claude，质量最好）
   - Google（Gemini，免费额度大）
   - Moonshot AI（Kimi，超长上下文）
   - DeepSeek（极致低价）
4. **输入 API Key**
5. **选择默认模型**
6. **完成配置**

**获取 API Key**：
- Anthropic: https://console.anthropic.com/
- Google: https://makersuite.google.com/app/apikey
- Moonshot: https://platform.moonshot.cn/
- OpenAI: https://platform.openai.com/api-keys

### 验证配置

```bash
openclaw models list                    # 查看已配置的模型
openclaw message send "你好，测试一下"   # 测试模型连接
```

### 快速切换模型

```bash
# 切换到 DeepSeek
openclaw config set agents.defaults.model.primary "deepseek/deepseek-chat"

# 切换到 Claude Haiku
openclaw config set agents.defaults.model.primary "maynor/claude-haiku-4-5-20251001"

# 重启生效
openclaw gateway restart
```

### 新手推荐配置

## 五、Gateway 网关配置（进阶）

### 配置文件位置
`~/.openclaw/openclaw.json`

### 访问控制配置

#### 限制 WhatsApp 访问
```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+15555550123", "+8613800138000"],
      "groups": {
        "*": {
          "requireMention": true
        }
      }
    }
  }
}
```

#### 配置群组提及规则
```json
{
  "messages": {
    "groupChat": {
      "mentionPatterns": ["@openclaw", "@小龙虾", "@助手"]
    }
  }
}
```

### 远程访问配置（Tailscale）

```bash
# 安装 Tailscale
brew install tailscale  # macOS
curl -fsSL https://tailscale.com/install.sh | sh  # Linux

# 启动并登录
sudo tailscale up

# 获取 Tailscale IP
tailscale ip -4

# 通过 Tailscale IP 访问
http://你的Tailscale-IP:18789/
```

### 完整配置示例

```json
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+8613800138000"],
      "groups": {
        "*": {
          "requireMention": true
        }
      }
    }
  },
  "messages": {
    "groupChat": {
      "mentionPatterns": ["@openclaw", "@助手"]
    }
  },
  "agents": {
    "defaults": {
      "model": {
        "primary": "deepseek/deepseek-chat",
        "fallbacks": ["anthropic/claude-haiku"]
      }
    }
  },
  "models": {
    "mode": "merge",
    "providers": {
      "deepseek": {
        "baseUrl": "https://api.deepseek.com",
        "apiKey": "sk-你的密钥",
        "auth": "api-key",
        "api": "openai-chat"
      }
    }
  }
}
```

**配置生效**：
```bash
openclaw gateway restart
```
