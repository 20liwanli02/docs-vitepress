# OpenClaw 是什么

## 一、概念

### 一句话介绍

**OpenClaw** 是一个**开源的 AI 智能体 Gateway 网关**，可以在本地部署 AI 助手，访问本地文件，并通过多个平台（飞书、企微、QQ 等）随时使用。

> 📌 **名称演变**：原名 Clawdbot → 过渡名 Moltbot → 最终定名 OpenClaw（2026年1月30日），三个名字本质是同一个项目。

### 核心能力
| 能力             | 说明                                           |
| ---------------- | ---------------------------------------------- |
| 🏠 **本地部署**   | 在自己的电脑上运行，数据隐私有保障             |
| 📁 **文件访问**   | 搜索、读取、编辑电脑上的文件                   |
| 🔌 **无限扩展**   | 通过 Skills（技能）系统扩展功能                |
| 💬 **多平台使用** | 支持飞书、企微、钉钉、QQ、Telegram、Discord 等 |
| 💰 **成本可控**   | 使用自己的 API，费用透明                       |

### 核心组件
1. **Gateway 网关** - 中央调度站，连接各聊天平台，管理会话和路由
   - 默认地址：`http://127.0.0.1:18789/`
   - 配置文件：`~/.openclaw/openclaw.json`
2. **AI 智能体** - 支持 Claude、GPT、Gemini、DeepSeek、Kimi 等模型
3. **Skills 技能系统** - 预装 49 个内置技能，涵盖文件管理、知识管理、自动化等
4. **ClawHub** - 技能市场，可下载和分享 Skills

## 二、为什么选择 OpenClaw

### 与在线 AI 的本质区别

| 特性         | OpenClaw      | ChatGPT/Claude 网页版 |
| ------------ | ------------- | --------------------- |
| 部署方式     | 本地/云端     | 在线服务              |
| 数据隐私     | ✅ 完全掌控    | ⚠️ 上传到服务器        |
| 本地文件访问 | ✅ 支持        | ❌ 不支持              |
| 系统操作     | ✅ 支持        | ❌ 不支持              |
| 功能扩展     | ✅ Skills 系统 | ❌ 固定功能            |
| 成本         | 按需付费      | 订阅制                |
| 多平台集成   | ✅ 支持        | ⚠️ 有限                |

**简单类比**：
- ChatGPT/Claude：像在网吧上网，方便但受限
- OpenClaw：像自己的电脑，自由但需要配置

### 五大核心优势

#### 1. 本地部署，保护隐私
- 敏感文件无需上传云端，完全在本地处理
- 适合处理财务报表、合同等机密文档

#### 2. 访问本地文件和系统
- 可直接搜索电脑上的所有文件
- 支持日历、备忘录、截图等系统操作
- **示例**："帮我找一下我电脑上的一张发票，里面详情是买了一个跑步机" → AI 秒找到并发送文件

#### 3. 可扩展的 Skills 生态
- 预装 49 个内置技能，开箱即用
- 技能涵盖：文件管理、知识管理、日程管理、自动化、工具类（截图、翻译、画图、视频生成）
- 可通过 ClawHub 下载更多技能（1715+ 个）

#### 4. 多平台支持
- 企业微信、钉钉、飞书、QQ、Telegram、Discord
- 在任何平台发消息，AI 都能响应

#### 5. 成本可控
| 服务                    | 月费         |
| ----------------------- | ------------ |
| ChatGPT Plus            | ~140 元      |
| Claude Pro              | ~140 元      |
| **OpenClaw + DeepSeek** | **5-30 元**  |
| **OpenClaw + Kimi**     | **10-50 元** |

💡 **省钱技巧**：使用国产大模型可节省 50%-70% 成本

## 四、适用场景

### ✅ 适合使用 OpenClaw 的场景

| 场景             | 具体应用                                                   |
| ---------------- | ---------------------------------------------------------- |
| **个人效率提升** | 整理文档资料、管理日程任务、自动化重复工作                 |
| **知识管理**     | 网页文章存档、GitHub 项目管理、论文笔记整理、跨设备同步    |
| **编程开发**     | 代码搜索理解、技术文档查询、开发环境配置、项目管理         |
| **团队协作**     | 项目管理自动化、文档协作优化、会议记录整理、团队知识库建设 |
| **创意工作**     | AI 绘画工作流、视频脚本生成、多语言翻译、数据分析自动化    |

### ❌ 不适合使用 OpenClaw 的场景

| 场景             | 替代方案                             |
| ---------------- | ------------------------------------ |
| **纯在线对话**   | ChatGPT 网页版（无需安装配置）       |
| **专业代码编辑** | Cursor、GitHub Copilot（集成度更高） |
| **移动端为主**   | ChatGPT App、Claude App（无需配置）  |
| **不想折腾**     | 在线 AI 服务（开箱即用）             |

### 💡 最佳实践组合
- **OpenClaw**：日常工作助手（本地文件、自动化、多平台）
- **Cursor**：代码编辑
- **ChatGPT**：移动端对话
