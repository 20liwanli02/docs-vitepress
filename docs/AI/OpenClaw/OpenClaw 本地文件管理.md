## 一、智能文件搜索

### 为什么需要智能搜索

| 传统搜索痛点                 | OpenClaw 智能搜索  |
| ---------------------------- | ------------------ |
| 忘记文件名，只记得内容       | ✅ 根据文件内容搜索 |
| 文件太多，不知道在哪个文件夹 | ✅ 跨文件夹智能搜索 |
| 需要同时搜索多种类型         | ✅ 理解自然语言描述 |
| 需要根据内容筛选             | ✅ 智能过滤和排序   |

### 搜索技巧

| 技巧                   | ❌ 不好的搜索        | ✅ 好的搜索                                       |
| ---------------------- | ------------------- | ------------------------------------------------ |
| **描述内容而非文件名** | 找一下 IMG_1234.jpg | 找一下我去年在海边拍的照片                       |
| **提供上下文**         | 找发票              | 找一下我上个月买跑步机的发票，应该在下载文件夹里 |
| **指定搜索范围**       | 找所有文件          | 在文档文件夹里找所有的PDF文件                    |
| **使用排序**           | 找图片              | 找最近修改的10张图片，按时间排序                 |

### 安全删除策略

| 步骤       | 操作                            |
| ---------- | ------------------------------- |
| **第一步** | 分析 - 识别可删除文件           |
| **第二步** | 备份 - 重要文件先备份           |
| **第三步** | 删除 - 移动到回收站而非直接删除 |

**保护重要文件**：
```
在清理之前，请确保以下文件夹不被删除：
- ~/Documents/重要文档
- ~/Projects
- ~/Photos
```

**可恢复删除**：
```
先把文件移动到回收站，而不是直接删除，
这样如果误删了还能恢复。
```
